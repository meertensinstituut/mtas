Statistics

To get statistics in Solr requests, besides the parameter to enable the Mtas query component, the following parameter should be used.

Parameter Value Obligatory
mtas.stats true yes

Using this parameter, it is possible to add statistics on positions, tokens and spans to the response on a request.
